# CBOX 数字脑语音助手

一个参考车机语音助手效果图实现的可交互网页。支持按住说话、真实麦克风声波、中文语音识别、语音唤醒、快捷指令、指令执行结果、简短问答和边界场景反馈。

## 本地运行

```bash
npm install
npm run dev
```

打开终端提示的本地地址，首次使用时允许浏览器访问麦克风。建议使用最新版 Chrome 或 Edge；浏览器必须运行在 `localhost` 或 HTTPS 环境中才能使用麦克风。

## 交互方式

- 按住中央按钮开始说话，松开后处理语音。
- 点击右侧“语音唤醒”，说“你好小智”后直接提出问题。
- 点击左侧快捷指令可完整演示指令和问答流程。
- 右侧可切换语音播报，也可播放三态动画演示。

## 支持的示例

- “扬声器音量调高到 80%”
- “打开张三患者的扫描图像”
- “如何做好医疗影像质控？”
- “今天有哪些待办？”
- “你是谁？”

本项目为纯前端演示，语音内容不会上传到自建服务器。语音识别能力由浏览器 Web Speech API 提供，具体可用性取决于浏览器和系统环境。
